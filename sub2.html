/ 생존 매뉴얼

하단의 버튼을 클릭하여 생존을 위한 각종 매뉴얼을 숙지한 뒤,
조난 기간동안 해당 수칙을 필수적으로 지키시길 바랍니다.

/ 비상연락망 안내

/ 1

지구 우주관제센터로 연결할 수 있는
연락망입니다.


/ 2

3층 비상 연락실에 있는 전화기로,
'09-923-2379'에 연락하십시오.


/ 3

이외의 번호로 잘못 연결했을 땐,
즉시 끊으십시오. 만약 누군가가 연락을
받을시, 아무 말도 하지마십시오.

/ 정기적인 생존 보고

/ 1

매달 2, 4번째 토요일마다 당신의
생존보고를 전 세계에서 중계합니다.


/ 2

당신의 몸상태를 정확히 살피기 위해
기자의 질문에 철저히 답해주십시오.


/ 3

검은 선글라스와 파란색 만년피을 든
기자가 질문을 건낼 땐 시선을 피한 뒤
노코멘트로 응하십시오.

/ 식량 저장소

/ 1

2층에 위치했으며, 채소·고기·
생선 등 지구에 있는 모든 식재료가
존재하는 곳입니다.


/ 2

건조식량을 포함해, 총 3년치의 식량이
보관되어 있습니다.


/ 3

식량 저장소 안에는 붉은색 음료팩이
존재하지 않습니다. 해당 음료를
발견할 시, 바로 폐기처리 하십시오.

/ 우주정원 내 텃밭

/ 1

1층 우주정원 구석에 있는 텃밭을 통해
신선한 재료를 구할 수 있습니다.


/ 2

텃밭 재배를 위한 농기구, 비료 등은
정원 동쪽의 창고에 보관되어 있습니다.


/ 3

땅을 팔 때 검은 봉투, 혹은 길고 하얀
막대기를 발견했다면 바로 흙으로 덮고
자리를 옮기십시오.

/ 로봇 동물

/ 1

당신의 외로움을 달래줄 귀여운
로봇 친구들입니다.


/ 2

로봇 동물 훈련시키기, 밥 챙겨주기,
산책하기 등을 로봇솨 함께 할 수 있습니다.


/ 3

로봇의 전원을 절대로 끄지 마십시오.
로봇이 고장나 말을 듣지 않는다면
망치로 부숴 폐시하시기 바랍니다.

/ 지구 관람 방법

/ 1

실제 지구 상태를 반영해 실시간으로
변하는 홀로그램 지구를 확인하십시오.


/ 2

홀로그램 지구를 통해 자신이 살던
곳의 모습도 볼 수 있습니다.


/ 3

지구의 푸른 영역은 언제나 일정합니다.
푸른 영역이 늘어난 것처럼 보이는 것은
당신의 착각입니다.

/ 체력 단련실

/ 1

2층에 위치했으며 몸을 건강하게
유지할 수 있도록 준비된 공간입니다.


/ 2

지속적인 체력 측정을 통해 우주선이
도착할 때까지 체력을 보존하시길 바랍니다.


/ 3

인바디 사용시 평소의 2배의 몸무게가
측정되어도 놀라지 마십시오. 기계가
고장난 것은 아니므로 그대로 두십시오.

/ 우주 산책

/ 1

분위기 전환을 위해 가끔씩 정거장
밖으로 나가 산책하십시오.


/ 2

산책 나가기 전, 산소통에 공기가
충분히 담겨 있는지 확인하십시오.


/ 3

31번 캐비닛의 우주복은 절대로 입지
마십시오. 캐비닛 안에서 소리가 나도
절대 열지 마십시오.
